Endless Pomodoro
I wanted a pomodoro timer that fits my vibe and my type of music, something calm and something that makes me focus. The combination of the background and the music is exactly as I wanted. I hope who ever uses this timer find it useful as I did!

Features:
1. beautiful background
2. calm music
3. 25 mins timer
4. 5 mins break

Instructions:
Start button: click to start the timer
Pause button: click to pause the timer. After pausing, click start button to resume
Music button: play music whenever you wish
Restart button: you can restart the timer anytime!
